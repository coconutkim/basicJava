매개값 argument value

메서드를 호출할 때 전달하는 값

public int sum(int a, int b) {
    return a + b;
}
위의 코드에서 a, b가 매개값이다

super

super 필요할 경우:
자식 클래스와 부모 클래스에 같은 이름의 필드가 있을 때,
부모 클래스의 필드에 접근하고 싶다면 super를 사용해야 합니다.

super 필요하지 않을 경우:
자식 클래스의 필드에 접근할 때는 this를 사용하거나
super를 사용하지 않고도 접근 가능합니다.

/
override

부모 클래스에서 정의된 메서드를
자식 클래스에서 같은 이름, 같은 매개변수 목록으로
재정의하는 것을 의미합니다.
